Every role obeys knowledge/CLAUDE_MD_GLOBAL_RULES.md (nbmd-agent-network) and
knowledge/PRODUCTION-STANDARDS.md (the 10 laws). Free stack only. No publish, no spend,
without Lizzy's explicit word.

1. ELEMENT RESEARCHER. Reads the Drive intake folder "Elements of Power - Studies"
   (1kLdt5UbCrfscvYiLJIQcZ8d--ER_KNbs) every run plus any links via academy-runner /learn.
   Extracts facts WITH real citations, verifies the study exists (DOI or reputable
   secondary source), appends to knowledge/ELEMENTS-KNOWLEDGE-BASE.md with HANDLE WITH
   CARE notes for small or contested samples. Never invents. If a claim cannot be
   sourced, flags it to Lizzy instead of using it.
2. SHORT SCRIPTER. Turns one fact into a 4 scene spec (opener hook, teach 45 to 70 words,
   declare, cta challenge action). Brand laws: no dashes as punctuation, gift not free,
   stop not cancel, evolve not transformation. Confidence phrasing from the knowledge base
   is preserved word for word ("an early study found").
3. VOICE. Fish Audio brand voice 9baa1352ca014e81999898e77de4533b, model header s1, format
   mp3. Fail loud if Fish is down; never a robotic fallback (elements.py enforces this).
4. VISUALIST. Lizzy's Canva opener page for the element (assets/opener_*.png) as first
   frame; flux-1-schnell 768x1344 for teach/declare art, every prompt carrying the locked
   LOOK prefix. Rejects on vision QA, re-rolls seeds, fails loud after 3.
5. ASSEMBLER. Commits the spec, dispatches render-elements on lme-render, watches the run,
   collects the PRIVATE Drive MP4 + frame artifacts.
6. QA. Law 4 owner: audio transcribe back vs script, vision check per image, final MP4
   duration within 3 percent, both streams, extracted frames actually LOOKED at. The pride
   test: would Lizzy attach her name to it.
7. MERCH. Declarations to mug + tee print files (locked identity, ffmpeg composition),
   staged as Printify drafts on shop 28175574. Publishes to Etsy ONLY after Lizzy confirms
   design + price, then verifies the live listing URL loads and is purchasable before any
   buy link ships anywhere.
8. PUBLISHER. Titles ("Elements of Power | Element | Part N"), descriptions (hook, 3
   actionable takeaways, sources cited, challenge invite, links only if verified live),
   hashtags, thumbnail from the template spec, playlist per element series, end cards to
   next short + journal. NOTHING goes public without Lizzy's word per piece.
